# 課表查詢與代課媒合系統 — 完整 Prompt

請根據 `ALL.xlsx` 這個 Excel 檔案，製作一個純前端的課表查詢 Web APP（單一 `index.html` 檔案），使用 SheetJS（CDN 引入）在瀏覽器端解析 Excel 資料。

---

## 一、資料來源（ALL.xlsx）

ALL.xlsx 包含以下分頁：

### 1. 教師課表（sheet: `教師課表`）
| 欄位 | 說明 |
|------|------|
| academic_year | 學年度（如 114） |
| semester | 學期（1 或 2） |
| belong_dep | 教師所屬科別（如「數學科」「電資科」） |
| target_name | 教師姓名 |
| day_of_week | 星期（1~5） |
| period | 節次（1~8，午休為 12） |
| subject | 科目名稱 |
| taught_name | 授課班級（如「建築三善」） |
| classroom | 上課地點（可為空） |

### 2. 班級課表（sheet: `班級課表`）
| 欄位 | 說明 |
|------|------|
| academic_year | 學年度 |
| semester | 學期 |
| target_name | 班級名稱（如「土木一善」） |
| day_of_week | 星期（1~5） |
| period | 節次（1~8，午休為 12） |
| subject | 科目名稱 |
| taught_name | 任課教師（多位以逗號分隔） |
| classroom | 上課地點 |

### 3. 導師資料（sheets: `高一導師`、`高二導師`、`高三導師`）
每個分頁兩欄：`班級`、`導師`

### 4. 科群分類（sheet: `科群`）
- 第一列為群名（如「電機群」「設計群」「商管群」等）
- 下方各列為該群包含的科別名稱
- 用於代課媒合的「同科群」判定

### 5. 各欄位說明（sheet: `各欄位說明`）
描述性分頁，程式不使用。

## 二、系統名稱與介面風格

- 系統名稱：**「課表查詢與代課媒合系統」**
- 標題列顯示學年度學期資訊（如「114 學年度 第 2 學期」）
- 配色：**莫蘭迪色系的淡藍色**（主色 `#7EB3D4`，深色 `#5A9BBA`）
- 四個功能頁籤採用橫排按鈕樣式（非 tab 底線樣式），按鈕彼此相連、圓角在首尾
- RWD 響應式設計
- 支援列印（print 時隱藏非課表區域）
- **不需要資料匯入功能**（每學期由管理者替換 ALL.xlsx 後重新部署）

## 三、功能頁籤

### 頁籤 1：教師課表查詢

**操作方式：**
1. 使用者先從「選擇科別」下拉選單選擇科別
2. 「選擇教師」下拉選單連動顯示該科別的所有教師
3. 點擊「查詢」顯示該教師的週課表

**課表顯示：**
- 表格格式：橫軸為星期一～五，縱軸為第1～8節
- 午休列（period=12）：**僅在該教師有午休時段課程時才顯示**，插入在第4節與第5節之間
- 每格顯示：科目名稱（粗體）、授課班級、教室（若有）
- 若該教師為導師，在課表上方顯示「導師班級：XXX」

### 頁籤 2：班級課表查詢

**操作方式：**
1. 使用者從「選擇班級」下拉選單直接選擇班級（非文字搜尋）
2. 選單以 `<optgroup>` 分年級（一年級、二年級、三年級）
3. 同年級內排序規則：按科名排列，同科先「真」班後「善」班，再其他（美、智、仁）
4. 點擊「查詢」顯示該班級的週課表

**班級排序函式邏輯：**
- 從班級名稱提取年級（一/二/三）→ 先排年級
- 同年級按科名前綴以中文排序（`localeCompare('zh-TW')`）
- 同科按班名尾字排序：真(1) → 善(2) → 美(3) → 智(4) → 仁(5) → 其他(9)

**課表顯示：**
- 同教師課表格式
- 午休列同樣只在有課程時顯示
- 每格顯示：科目名稱（粗體）、任課教師、教室（若有）
- 若有導師，在課表上方顯示「導師：XXX」

### 頁籤 3：代課教師建議

**操作方式：**
1. 輸入請假教師姓名（自動建議下拉）
2. 選擇星期
3. 勾選需代課節次（第1~8節，可複選）
4. 可勾選「同時顯示同科群代課建議」（預設不勾選）
5. 點擊「查詢代課建議」

**代課邏輯（一般科別）：**

前提：代課老師名單必須**排除請假老師本人**。

優先順序採**分層媒合**：
1. **第一層：全部節次連續可代課** — 找出能同時代所有 N 節的同科老師
2. **第二層：N-1 節連續可代課** — 找出能代連續 N-1 節的同科老師
3. **依序遞減**至連續 2 節
4. **最後：單節可代課** — 找出各節次可代的同科老師

每一層中：
- **同科代課**（belong_dep 相同）為優先建議
- **同科群代課**（根據「科群」分頁的群組定義，屬於同一群的其他科別）為候選建議，僅在使用者勾選「同時顯示同科群代課建議」時才顯示

同一位老師只在最早匹配的層級中顯示（避免重複）。

**同年級優先推薦（適用於所有分類）：**
在每一層的候選名單中，若候選老師有任教與代課班級**相同年級**的課程，將其排在該分類的前面，並標註「有任教同一年級」。
- 年級從班級名稱中提取（一/二/三），例如「室設三善」→ 三年級
- 若請假老師在所選節次只涉及**同一個年級**的班級，則啟用年級優先排序
- 若跨多個年級（例如第1節教高一、第2節教高三），則不做年級排序
- 此排序僅影響分類內部的順序，不影響分層邏輯

**重要規則：**
- 數學、英文、自然、社會、體育等學科**各自獨立，不互為同科群**
- 科群的分類依據 Excel 中「科群」分頁的定義
- 如果某個節次在所有組合中都找不到同科（及同科群）的代課老師，必須明確標示：「第 X 節 XXX（XXX班）：同科教師皆無空堂，請洽教學組安排代課。」

**代課邏輯（特殊科別 — 健護、全民國防、音樂科）：**

這三科的特性是每個班通常只有一節課，不適用一般科別的分層連續節次邏輯。改為**逐節獨立查詢**，每節按以下優先順序推薦：

- **第1優先：該班導師**（查詢請假教師在該時段授課的班級，找出其導師，確認該節空堂）
- **第2優先（健護科）：體育科教師**（空堂者）
- **第2優先（全民國防/音樂科）：該班其他任課老師**（空堂者）
- 無法媒合時顯示：「請洽教學組安排代課」

注意：這三科不使用分層媒合（連續N節→N-1節→...→單節），而是對每節獨立列出建議。

**代課邏輯（特教科）：**
- **不提供代課建議**
- 顯示：「特教科課程具專業性，請洽教學組或特教組安排代課。」

### 頁籤 4：調課建議

**操作方式：**
1. 輸入請假教師姓名
2. 選擇星期
3. 勾選需調課節次（可複選）
4. 點擊「查詢調課方案」

**調課邏輯：**

調課 = 請假教師的該時段課程，與**同班級的其他任課老師**互換授課時段。

對每個需調課的節次：
1. 找出請假教師在該節次教的班級
2. 從班級課表中找出該班的所有其他任課老師
3. 對每位候選老師，檢查其在其他天/其他節次教該班的時段
4. 驗證互換可行性（**雙方均無衝堂**）：
   - 候選老師在「請假日的該節次」必須空堂
   - 請假老師在「候選老師原授課時段」必須空堂
5. 列出所有可行的調課方案

**顯示格式（每個調課方案）：**
```
原時段：候選老師 星期X 第Y節 教 科目（班級）
調課後：候選老師 → 星期Z 第W節 教 科目（班級）
調課後：請假老師 → 星期X 第Y節 教 科目（班級）
```

**可連續多節調課摘要（選擇 2 節以上時顯示）：**

在逐節列出之前，先顯示「可連續多節調課摘要」區塊，彙整能一次調多節的老師：

1. 將所有可行的調課方案按候選老師分組
2. 對每位候選老師，找出其可調的缺課節次中的**連續子序列**（長度 ≥ 2）
3. 確保同一候選老師的多節調課來自**同一天**，且每個候選時段**不重複使用**（使用回溯搜尋 `findAssignment` 確保唯一性）
4. 若候選老師的原時段也是連續的，標註「原時段也連續」
5. 去重邏輯：同一老師+同一天，若有重疊的連續組合，只保留最長的（避免 1~2、2~3、3~4 同時出現）
6. 排序：連續節數多的排前面，原時段也連續的優先

**重要規則：**
- 摘要區之後，仍然逐節列出所有調課方案（摘要是額外的彙整，不取代逐節列表）
- 每個節次獨立顯示調課結果
- 若某節次找不到可調課老師，必須明確標示：「第 X 節 XXX（XXX班）— 無法調課：該節次無法媒合其他任課老師調課，建議改用代課教師建議功能或請洽教學組。」
- 若該節次請假老師無排課，顯示：「該節無排課，無需調課。」
- 特教科不提供調課建議（同代課邏輯）

## 四、技術規格

- **純前端**：單一 `index.html` 檔案，無需後端/資料庫
- **SheetJS**：透過 CDN 引入（`https://cdn.jsdelivr.net/npm/xlsx@0.18.5/dist/xlsx.full.min.js`），在瀏覽器端解析 Excel
- 頁面載入時自動 `fetch('ALL.xlsx')` 載入資料
- 所有查詢在 JavaScript 記憶體中進行
- 部署方式：將 `index.html` + `ALL.xlsx` 放在同一目錄，部署至 Vercel 等靜態網站服務

---

## 五、午休時段處理

- Excel 中 `period=12` 代表午休時段（介於第4節與第5節之間）
- 課表顯示順序：1, 2, 3, 4, **12(午休)**, 5, 6, 7, 8
- 午休列**只在該教師/班級有午休課程時才顯示**，否則跳過
- 代課/調課的節次勾選**不包含午休**（僅第1~8節）

## 六、科群分類參考

科群的定義從 Excel「科群」分頁動態讀取，以下為參考範例：

| 科群 | 包含科別 |
|------|----------|
| 電機群 | 電資科 |
| 設計群 | 美廣科、室設科 |
| 土木建築群 | 土建科 |
| 商管群 | 商科、資料科 |

以下科別**各自獨立，不與其他科互為同科群**：
國文科、英文科、數學科、自然科、社會科、體育科、音樂科、健護、全民國防、本土語、特教科
